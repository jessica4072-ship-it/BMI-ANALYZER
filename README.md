# BMI-HEALTH ANALYZER
A small Python project that can calculate BMI and give a basic health suggestions and to spread awareness about fitness and lifestyle.

## About
This is a small Python project I have created to calculate BMI using height and weight. I chose this project because many students ignore their health in daily life.

## Problem
People are often not aware of their body weight condition and related health risks and often ignore it.

## Solution
The program calculates BMI and classifies the user into different health categories.

## Features
- It performs BMI calculation
- It displays Health category
- It also provides basic health advice

## How to Run
using file : Python main.py
